---
title: 非同期処理中の振る舞い
---

非同期処理を実行している際の待機情報を表す等の方法を説明していきます。
ここでの振る舞いとして、利用者（ユーザ）へ以下が分かるようにする。

- 処理中であること
- UIで操作が制限されていること

## 非同期処理中を表す実装

非同期処理中であることを利用者が分かるようにすることを目的に処理中を表す実装を説明する。

## 動作確認

非同期処理中の制御が実施出来ていることを確認する方法と期待値を説明する。
待機処理（非同期処理中）を確認するための方法も説明する。

タイムアウト等の異常系は、その点はエラーハンドリングの学習コンテンツで
学ぶこととする。
応答を遅らせる方法も説明する。
