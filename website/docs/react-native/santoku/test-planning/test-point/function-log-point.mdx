---
title: ログ出力に関する観点
---

機能テストに関するログ出力のテスト観点をまとめています。
それぞれのテスト観点の実施方法の種別を以下に列挙しています。

- (A) [自動テスト]Jestを利用して、作成したテストコードで自動テストを実行して確認
- (B) [手動テスト]Android、iOS／iPadOS毎にエミュレータ／シミュレータ
- (C) [手動テスト]実機（テスト対象機種分）

## ログ出力

| NO | 詳細 | 観点 | 補足 | テスト方法 |
| --- | --- | --- | --- | --- |
| 1 | ユーザ操作 | ユーザ操作に伴う処理中に正常系のログ出力が設計されている場合、設計通りにログが出力されていること | - | A |
| 2 | ユーザ操作 | ユーザ操作に伴う処理中にエラーが発生した場合、方式設計のエラーハンドリング方針に沿ってログが出力されていること | - | A |
| 3 | ユーザ操作以外のイベント処理  | 初期データ読み込みなどの処理中に正常系のログ出力が設計されている場合、設計通りにログが出力されていること | - | A |
| 4 | ユーザ操作以外のイベント処理  | 初期データ読み込みなどの処理中にエラーが発生した場合、方式設計のエラーハンドリング方針に沿ってログが出力されていること | - |  |
