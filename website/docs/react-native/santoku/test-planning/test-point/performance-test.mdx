title: 性能テスト
---

性能テストに関するテスト観点をまとめています。
性能テストでは、性能に関する事項の[応答時間について](../../requirements/non-functional/performance.mdx#応答時間について)と[CPU・メモリ使用率について](../../requirements/non-functional/performance.mdx#cpuメモリ使用率について)で
定めた性能目標を満たしているかどうかを確認します。
それぞれのテスト観点毎の検証方法は、[テスト方法](function-test.mdx#テスト方法)にまとめています。

## テスト観点

| NO | 詳細 | 観点 | 補足 | テスト方法（※1） |
| :--- | :--- | :--- | :--- | :--- |
| 1 | 応答時間 | ページ読み込みなどレンダリングにかかる時間が目標値以内であること | - | C |
| 2 | 応答時間 | ページ内でのスライドアニメーション動作などにかかる時間が目標値以内であること | - | C |
| 3 | CPU・メモリ使用率 | 何かしらの操作によって急激に上昇し、以降低下しない事象が発生しないこと | - | C |
| 4 | CPU・メモリ使用率 | 操作中緩やかに上昇するが、低下することなく上昇し続け高止まりする事象が発生しないこと | - | C |

（※1）性能テストは、Android、iOS／iPadOS毎に[テスト端末バリエーション](../../test-planning/test-variation.mdx)の中から最も性能が低い機種を選択します。
