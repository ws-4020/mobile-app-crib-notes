title: 性能テスト
---

性能テストに関するテスト観点をまとめています。
性能テストでは、非機能要件の[性能に関する事項](/react-native/santoku/requirements/non-functional/performance.mdx)で
定めた性能目標を満たしているかどうかを確認します。

## テスト観点

| NO | 詳細 | 観点 | 補足 | テスト方法 |
| --- | --- | --- | --- | --- |
| 1 | 応答時間 | ページ読み込みなどレンダリングにかかる時間が目標値以内であること | - | C（※1） |
| 2 | 応答時間 | ページ内でのスライドアニメーション動作などにかかる時間が目標値以内であること | - | C（※1） |
| 3 | CPU・メモリ使用率 | 何かしらの操作によって急激に上昇し、以降低下しない事象が発生しないこと | - | C（※1） |
| 4 | CPU・メモリ使用率 | 操作中緩やかに上昇するが、低下することなく上昇し続け高止まりする事象が発生しないこと | - | C（※1） |

（※1）性能テストは、全て実機で実施します。
Android、iOS(iPadOS)毎に[テスト端末バリエーション](/react-native/santoku/test-planning/test-variation.mdx)の中から最も性能が低いものを選択します。
