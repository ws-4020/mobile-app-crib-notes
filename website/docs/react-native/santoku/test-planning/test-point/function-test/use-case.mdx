---
title: ユースケースに関する観点
---

このアプリの利用シナリオをベースに網羅的なテストを実施します。
以下の観点より、テストとして必要なユースケースの観点を詳細化して決定します。
それぞれのテスト観点毎の検証方法は、[テスト方法](../function-test.mdx#テスト方法)にまとめています。

| ユースケース | 内容 | 例 | テスト方法 |
| :--- | :--- | :--- | :--- |
| 事前条件 | このアプリの基本となる利用フローを検証する条件 | 利用者が既にこのアプリをダウンロード済であり、アカウントも作成済 | B、C |
| トリガー | それぞれのユースケースを開始するための条件 | チームへ参加する招待コードの受け取り | B、C |
| 基本フロー | 利用シナリオを踏まえた正常系ルートの動作 | 時間割テンプレートを作成（編集含む）し、当日の時間割を作成して、時間割を開始 | B、C |
| 代替フロー | 例外系の動作 | チームへ招待する招待コードの有効期限切れ | B、C |
| 事後条件 | 期待する結果 | 当日の時間割を開始して、このアプリの利用が開始できていること | B、C |
