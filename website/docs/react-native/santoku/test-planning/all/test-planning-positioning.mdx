## テスト計画書の目的

テスト計画書の目的は開発するサンプルアプリケーションのテストを実施するにあたり、それぞれのテストの実施範囲、観点、内容を明確化することで、
プログラミング・単体テスト、結合テストにおいて必要な品詞を担保することを目的とします。

## テスト計画書の位置付け

画像を入れる

- 全体テスト計画書
  - 単体テスト計画書
    - 単体テストコード
    - 単体テストケース、データ
    - 単体テストエビデンス
  - 結合テスト計画書
    - 結合テストケース、データ
    - 結合テストエビデンス
