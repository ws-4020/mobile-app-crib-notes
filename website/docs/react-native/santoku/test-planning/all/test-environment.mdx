## テスト対象機種

網羅的なテスト実施を目的にテスト対象機種を以下の選定方針に従って、準備します。

### Androidのテスト対象機種

Androidは多くのメーカーが多種多様な機種をリリースしています。なので今回のアプリ開発時点から4年前までの期間の中でシェア（利用度合い）の高いメーカーの中から機種を選定します。

:::note info
サンプルアプリケーションであることと、汎用的に参考にしていただきたい考えから敢えて具体的なメーカー名や機種名は伏せて記載しています。
:::

シェアの高いメーカーとGoogleからリリースされているローエンド、ミドルレンジの機種を選定します。

### iOS（iPadOS）のテスト対象機種

iPhoneとiPadそれぞれの機種でシェア（利用度合い）の高い以下の機種をテスト対象端末として準備します。実機で準備が難しいものはシミュレータで準備します。

- iPhone

    iPhone 7、iPhone 8、iPhone 6（※1）

- iPad

    iPad Air、iPad、iPad Pro（※1）

（※1）左からシェアの高い機種を列挙。

## テスト対象OSバージョン

テスト対象機種と同様の目的で、テスト対象OSバージョンを以下の選定方針に従って、準備します。

### Androidのテスト対象OSバージョン

シェアの高いOSバージョンと開発時点から4年前までの期間の中でリリースされたOSバージョンの中でこのアプリに影響のある機能追加や変更があったバージョンをテスト対象OSバージョンとして選定します。
選定対象となるOSバージョンのバリエーションは、Android 8.0 ～ 11になります。

| OSバージョン | 選定理由 |
| :--- | :--- |
| Android 11 | ユーザ使用率の高いOSバージョンのため  |
| Android 10 | ダークモードが利用可能となったバージョンであり、このアプリに影響があるため |

### iOS（iPadOS）のテスト対象OSバージョン

ユーザ使用率の高いOSバージョンと、シェア3%未満に入らないOSバージョンをテスト対象OSバージョンとして選定します。
選定対象となるOSバージョンのバリエーションは、iOS（iPadOS）13 ～ 14になります。

| OSバージョン | 選定理由 |
| :--- | :--- |
| iOS 14 | ユーザ使用率の高いOSバージョンのため  |
| iOS 13 | シェア3%未満に入らないOSバージョンのため |
| iPadOS 14 | ユーザ使用率の高いOSバージョンのため  |
| iPadOS 13 | シェア3%未満に入らないOSバージョンのため |

以上までのテスト対象機種とテスト対象OSバージョンの組み合わせで網羅できるテスト環境を準備します。詳細なバリエーションについては、[テスト環境](test-environment)に定義します。
