---
title: 出力結果に関する観点
---

機能テストに関する出力結果のテスト観点をまとめています。
それぞれのテスト観点毎の検証方法は、[テスト方法](../functional-testing.mdx#テスト方法)にまとめています。

## テキスト表示

| NO | 詳細 | 観点 | 補足 | テスト方法 |
| --- | --- | --- | --- | --- |
| 1 | 固定文字列 | 仕様通りの文章が表示されていること | - | B |
| 2 | 外部データ | 仕様通りの文章が表示されていること（別の外部データが誤って表示されていないこと） | - | B |
| 3 | 外部データ | 外部データに含まれる改行と同じ位置で画面上でも改行されて表示されていること | - | B |
| 4 | 外部データ | 外部データをマスキングして表示する設計の場合、マスキングされた値が表示されていること | - | B |
| 5 | 外部データ | サロゲートペア文字を許容する場合、サロゲートペア文字が表示されていること | 絵文字などを表示する | B |
| 6 | フォーマット | 表示内容について、設計で指定されたフォーマット通りに表示されていること | - | B |

## 表示項目（一覧）

| NO | 詳細 | 観点 | 補足 | テスト方法 |
| --- | --- | --- | --- | --- |
| 1 | リスト表示 | デフォルトのソート順が設計で定められている場合、仕様通りのソート順で表示されていること | - | B |
| 2 | リスト表示 | 表示されたデータのソート順を変更する機能が存在する場合、実行すると指定されたソート順にデータがソートされて表示されること | - | B |
| 3 | リスト表示 | 表示データの件数が一覧の表示可能件数を超えた場合、ページングまたはスクロールによる自動追加読込み（無限スクロール）等によって続きが表示できること | - | B |
| 4 | リスト表示 | 表示データの取得に失敗した場合、仕様通りの内容が表示されること | - | B |

## 入力項目

| NO | 詳細 | 観点 | 補足 | テスト方法 |
| --- | --- | --- | --- | --- |
| 1 | テキスト入力 | テキスト入力欄に対応するラベルをタップすると、テキスト入力欄にフォーカスが移動すること | - | B |
| 2 | テキスト入力 | パスワード等の機密情報の入力欄は、入力内容がアスタリスクなどの記号に置き換えた形で表示されていること | - | B |
| 3 | テキスト入力 | 入力完了後のフォーカス移動が仕様で定められている場合、入力後に仕様通りフォーカスが移動すること | - | B |
| 4 | テキスト入力 | 現在の入力文字数を表示する場合、サロゲートペア文字や異体字を含む文字列を入力しても正しい入力文字数が表示されること | - | B |
| 5 | テキスト入力 | テキスト入力の要素ごとに、仕様通りのキーボード種別で表示されていること | - | B |
| 6 | テキスト入力 | 入力済みの項目をクリアする仕様がある場合、クリアを実行することで対象の項目がクリアされていること | - | B |
| 7 | テキスト入力 | プレースホルダーに表示されている内容が仕様通りに表示されていること | - | B |
| 8 | セレクトボックス | 表示される選択肢が動的に変化する場合、変化する条件に応じて仕様通りの選択肢のみが表示されていること | - | B |
| 9 | ラジオボタン | 表示される選択肢が動的に変化する場合、変化する条件に応じて仕様通りの選択肢のみが表示されていること | - | B |
| 10 | チェックボックス | 表示される選択肢が動的に変化する場合、変化する条件に応じて仕様通りの選択肢のみが表示されていること | - | B |
| 11 | ファイルアップロード | ファイルアップロードするUI部品は、アップロード中のステータスが分かるようなUIを表示していること | 例：プログレスバー | B |
| 12 | ファイルアップロード | ファイルアップロードするUI部品は、アップロードが完了したら仕様通りの結果が表示されていること | - | B |

## 出力エラー

| NO | 詳細 | 観点 | 補足 | テスト方法 |
| --- | --- | --- | --- | --- |
| 1 | メッセージ | エラー発生時のエラーメッセージが仕様通りの内容となっていること | - | B |
| 2 | メッセージ | 単項目バリデーションの結果、エラー条件に該当した場合、該当する箇所へ仕様通りのエラーメッセージが表示されていること | - | B |
| 3 | メッセージ | 相関バリデーションの結果、エラー条件に該当した場合、該当する箇所へ仕様通りのエラーメッセージが表示されていること | - | B |
