---
title: 同一構成のエミュレータの利用
---

同一構成のエミュレータをメンバー全員で共有する際の手順です。

## 前提

複数人でのテスト実施を想定しています。複数人でテストを実施する場合、エミュレータの構成をメンバー全員で共有しておかないと、誤った環境でテストを進めてしまう可能性が高くなるためです。

## Androidエミュレータ

### プロファイルの作成と共有

実施対象者はテスト実施メンバーのうち1名のみです。

1. 仮想デバイス作成画面を開く

   ```text
   Android Studio ＞ AVD Manager ＞ Create Virtual Device...
   ```

2. `New Hardware Profile`をクリックし、プロファイル作成画面を開く
3. テスト対象のプロファイルに従い、入力する
4. `Finish`をクリックし、プロファイル作成完了
5. 仮想デバイス作成画面で、作成したプロファイルを右クリックから`Export`を選択し、`xml`ファイルとしてエクスポートする
6. テストに必要な構成分、`2～5`を繰り返す
7. エクスポートしたプロファイルをチーム全員が参照できる場所に格納し、共有する

### プロファイルのインポート

実施対象者は、上述のプロファイル作成者以外のテスト実施メンバーです。

1. 仮想デバイス作成画面を開く

   ```text
   Android Studio ＞ AVD Manager ＞ Create Virtual Device...
   ```

2. `Import Hardware Profiles`をクリックする
3. 共有されたプロファイルを選択（複数選択可）する
4. `OK`をクリックし、インポート完了

:::info
プロファイルに`OSバージョン`は含まれません。そのため、必要に応じて作成／インポートしたプロファイルを元にOSバージョンごとのエミュレータ環境を作成してください。
:::

### エミュレータの作成

1. 仮想デバイス作成画面を開く

   ```text
   Android Studio ＞ AVD Manager ＞ Create Virtual Device...
   ```

2. テスト対象のプロファイルを選択し、`Next`をクリックする
3. もしテスト対象のOSバージョンがダウンロードできてなければ、ダウンロードする
4. テスト対象のOSバージョンを選択し、`Next`をクリックする
5. `Finish`をクリックし、作成完了

## iOSシミュレータ

`iOS`は`Android`と違い、プロファイルを作成する必要がありません。実施対象者はテストメンバー全員です。

### OSバージョンのダウンロード

テスト対象のOSバージョンが不足している場合は、対象OSバージョンのダウンロードが必要です。

1. シミュレータのOS一覧画面を開く

   ```text
   Xcode ＞ Preference ＞ Components
   ```

2. 必要なOSを選択し、ダウンロードする

### シミュレータの作成

1. Xcodeで開発プロジェクトを開く
2. 画面上部に表示されている機種名をクリックする
3. `Add Additional Simulators`をクリックする
4. 画面左下の`+`をクリックする
5. テストに必要な機種とOSを選択する
6. `Create`をクリックして、作成完了
