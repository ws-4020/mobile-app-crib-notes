## メッセージキーの体系


### ラベル

```
'パスワード'
'ホーム'
'最大{0}件'
```

### バリデーション

```validation.required
validation.min
validation.max
```

### 基盤部品で使うメッセージ

```
'fw.コンフリクト': '処理が競合しました。'
```

### アプリで使うメッセージ

```
'app.登録完了': '登録しました。'
'app.task.期限超過{0}件エラー': '期限を超過したタスクが{0}件です。確認してください。'
'app.task.タスク登録完了': 'タスクを登録しました。'
```
