---
title: ログの利用
---

このアプリケーションでは、リリース後の障害解析を目的として、エラーレベルのログをFirebase Crashlyticsに送信します。

エラーレベルのログには、以下の項目を含めることができます。

- メッセージ
- エラーコード

Firebase Crashlyticsに記録されるエラーログは、エラーコードを基準にグループ化されます。そのため、エラーコードにはカテゴライズしたい単位で値を設定してください。

以下は、エラーログを送信する例です。`log.error`の第一引数がメッセージ、第二引数がエラーコードになります。

```typescript title="起動IDの取得に失敗した場合"
log.error('Failed to get launchedId.', 'LaunchedIdGettingFailure');
```

```typescript title="メッセージのロードに失敗した場合"
log.error('Failed to load message.', 'BundledMessagesLoadingFailure');
```

:::caution
このアプリケーションでは、機密情報のログ出力を禁止しています。そのため、ログに含めるメッセージやエラーコードには機密情報を含めないように注意してください。
:::
