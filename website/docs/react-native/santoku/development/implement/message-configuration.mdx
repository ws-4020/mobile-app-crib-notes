---
title: メッセージの定義と取得
---

このアプリケーションでは、アプリ内にバンドルしたメッセージ定義を使用します。

## メッセージの定義

メッセージを新たに定義する場合は、メッセージ定義ファイル（`src/generated/BundledMessages.ts`）に定義を追加します。

```diff typescript title="BundledMessages.ts"
export const bundledMessages = {
  /* 中略 */
+ ユーザー名: 'ユーザー名',
  パスワード: 'パスワード',
  ホーム: 'ホーム',
  '最大{0}件': '最大{0}件',
  'ユーザー名とパスワードを入力してください。': 'ユーザー名とパスワードを入力してください。',
  /* 中略 */
} as const;
```

## メッセージの取得

メッセージを取得する場合は、`src/framework/message/Message.ts`を使用します。

```typescript jsx title='メッセージの取得'
import {m} from 'framework';

console.log(m('最大{0}件', '3'));  // => 最大3件
```
