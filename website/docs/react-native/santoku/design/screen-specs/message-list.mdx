このアプリ内で表示するメッセージの一覧を記載します。<br />
メッセージの種類とキーについては[メッセージの管理](/react-native/santoku/development/design/message-configuration)を参照。

## ラベル

| メッセージキー              | メッセージ                                                                     |
|:--------------------------|:-----------------------------------------------------------------------------|
| 利用規約                   | 利用規約                                                                       |
| 同意                       | 同意                                                                          |

## メッセージ

| プレフィックス   | メッセージキー         | メッセージ                                                                 |
|:---------------|:--------------------|:------------------------------------------------------------------------|
| app.terms      | 取得エラー           | 通信エラーが発生し、利用規約の情報取得に失敗しました。ネットワーク環境を確認してください。  |
| app.terms      | 同意エラー           | 通信エラーが発生し、利用規約の同意に失敗しました。ネットワーク環境を確認してください。     |
