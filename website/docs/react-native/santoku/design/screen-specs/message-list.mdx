このアプリで表示するメッセージの一覧を記載します。<br />
メッセージの種類とキーについては[メッセージの管理](/react-native/santoku/development/design/message-configuration)を参照。

## ラベル

| メッセージキー              | メッセージ                                                                     |
|:--------------------------|:-----------------------------------------------------------------------------|
| 利用規約                   | 利用規約                                                                       |
| 同意                       | 同意                                                                          |

## メッセージ

| プレフィックス   | メッセージキー         | メッセージ                                                                 |
|:---------------|:--------------------|:------------------------------------------------------------------------|
| app            | ネットワークエラー     | 通信エラーが発生し、{0}に失敗しました。ネットワーク環境を確認してください。  |
| app            | HTTPエラー           | {0}に失敗しました。時間をおいて再度お試しください。    |
| app.terms      | 有効な利用規約の取得エラー     | 利用規約の取得に失敗しました。画面をリロードして再度お試しください。 |
| app.terms      | 有効な利用規約の同意エラー     | 利用規約の同意に失敗しました。ネットワーク環境を確認し、再度お試しください。 |
