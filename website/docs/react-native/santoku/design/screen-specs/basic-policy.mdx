アプリの画面全体を通じて、基本方針となる事項を記載します。<br />
画面の仕様は、非機能要件の「ユーザビリティおよびアクセシビリティに関する事項」（TODO:リンク張る）に準拠します。

## 使用するライブラリ

サンプルアプリケーションでは、UIライブラリとしてReact Native Elementsを使用します。
以下のUIについては、React Native Elementsの標準仕様に準拠いたします。

- スクロールバー
- インジケータ

## インジケータについて

通信中など、何かしら処理中であることをユーザに示す場合、画面上にインジケータを表示します。
ユーザの操作をできるだけ妨げないよう、表示領域は極力限定します。
:::note
以下は一例となります。

  - 画像やリストの遅延読み込みの場合： 画像やリストの表示領域にのみインジケータを表示<br />
  - ボタン押下による通信の場合：ボタン内のみインジケータを表示
:::

## デザインについて

このアプリではレスポンシブデザインを採用します。
端末の画面サイズごとの挙動は、プロトタイプにて確認します。

## プロトタイプについて

画面設計後、画面の挙動を確認するためのプロトタイプを作成いたします。
以下の点についてはプロトタイプで確認するため、設計書では割愛します。

- スクロール領域
- 端末の画面サイズによる可変/固定となる領域
