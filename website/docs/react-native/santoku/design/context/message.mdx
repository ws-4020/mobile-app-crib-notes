---
title: メッセージコンテキスト
---

## 概要

アプリ内で表示するメッセージを集約管理し、将来的な多言語対応を容易に行えるようにするコンテキストです。
詳細は、[メッセージ管理](../application-architecture/message-configuration/overview)を参照してください。

## 提供するインタフェース

- メッセージのロード
  - 指定したキーに対応する、アプリ内にバンドルされたメッセージを取得します
  - 詳細は、[メッセージのロード](../application-architecture/message-configuration/load-message)を参照してください

## 管理対象データとライフサイクル

管理対象となるメッセージは、アプリ内にバンドルする形で提供されます。
データの追加・更新・削除は、アプリを再ビルドすることで行います。
データの詳細は、[管理対象のメッセージ](../application-architecture/message-configuration/manage-message)を参照してください。
