---
title: メッセージコンテキスト
---

## 概要

アプリ内で表示するメッセージを集約管理し、将来的な多言語対応を容易に行えるようにするコンテキストです。
詳細は、[メッセージ管理](../../application-architecture/message-configuration/overview.mdx)を参照してください。

## 提供するインタフェース

- メッセージのロード
  - アプリ内のバンドルされたメッセージをロードします
- メッセージの取得
  - 指定したキーに対応するメッセージを取得します

詳細は、[メッセージのロード](../../application-architecture/message-configuration/load-message.mdx)を参照してください。

## 管理対象データ

管理対象となるメッセージは、アプリ内にバンドルする形で提供されます。
データの追加・更新・削除は、アプリを再ビルドすることで行います。
データの詳細は、[管理対象のメッセージ](../../application-architecture/message-configuration/managed-message.mdx)を参照してください。
