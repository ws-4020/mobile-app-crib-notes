## 管理対象のメッセージと管理対象外のメッセージ

管理対象とするメッセージを使用パターン毎に分類したリストを次に示す。

- 画面の表示項目
  - ラベル
  - 画面タイトル
  - 入力テキスト内のプレースホルダー
  - 操作ガイダンス
  - 入力エラーメッセージ
    - 単項目エラー
    - 相関項目エラー
- ダイアログ
  - 確認メッセージ
- スナックバー（※1）
  - 情報メッセージ
  - 警告メッセージ
  - エラーメッセージ

（※1）APIのレスポンスメッセージはそのまま用いず、処理結果に応じてモバイル端末側で適切なメッセージを表示。

:::info
APIのレスポンスメッセージをそのまま画面に表示する方針もあります。
どちらを採用するかは、それぞれのシステム背景により判断してください。
:::

一方で、管理対象外とするメッセージは次の通り。

- ログ
  - ログメッセージ
- アクセス許可ダイアログ
  - カメラやマイク、位置情報などを利用する際に表示するアクセス許可ダイアログのメッセージ
