## 管理対象のメッセージ

管理対象とするメッセージは次の通り。

- 画面の表示項目
  - ラベル
  - 画面タイトル
  - プレースホルダー
  - 操作ガイダンス
- ダイアログ
  - 確認メッセージ
- 結果メッセージ[^1]
  - バリデーションメッセージ
  - トーストメッセージ
  - エラーメッセージ
- プッシュ通知
  - ローカル通知メッセージ

[^1]: APIのレスポンスメッセージを結果メッセージとして表示するかどうかはシステムの背景による。

一方で、管理対象外とするメッセージは次の通り。

- ログメッセージ
- アクセス許可時に表示するメッセージ
  - カメラやマイク、位置情報などを利用する際に表示するアクセス許可ダイアログのメッセージ
- プッシュ通知
  - リモート通知メッセージ
