## 管理対象のメッセージ

管理対象とするメッセージを使用パターン毎に分類したリストを次に示す。

- 画面の表示項目
  - ラベル
  - 画面タイトル
  - 入力テキスト内のプレースホルダー
  - 操作ガイダンス
- ダイアログ
  - 確認メッセージ
- 結果メッセージ[^1]
  - 入力エラーメッセージ
    - 単項目エラー
    - 相関項目エラー
  - トーストメッセージ
    - 情報メッセージ
  - エラーメッセージ
    - 警告メッセージ
    - 業務エラーメッセージ
    - システムエラーメッセージ
- プッシュ通知（ローカル通知）
  - 通知メッセージ

[^1]: APIのレスポンスメッセージは結果メッセージとして用いず、処理に応じてモバイル端末側で適切なメッセージを表示する。APIのレスポンスメッセージをそのまま画面に表示するかどうかは、それぞれのシステム背景により判断。

一方で、管理対象外とするメッセージは次の通り。

- ログ
  - ログメッセージ
- アクセス許可ダイアログ
  - カメラやマイク、位置情報などを利用する際に表示するアクセス許可ダイアログのメッセージ
- プッシュ通知（リモート通知）
  - 通知メッセージ
