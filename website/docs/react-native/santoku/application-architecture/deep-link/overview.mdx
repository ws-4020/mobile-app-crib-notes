### URLの形式

ディープリンクの形式はURLのドメインとアプリケーションの設定を一致させたURLであればアプリに遷移できます。

URLの形式としては通常のURLと短縮URLがあります。通常のURLだと非常に長くなる[^1]ため、短縮URLを採用します。

[^1]: `https://関連付けするドメイン/?link=https://アプリケーションが受け取るURL/app?アプリのパラメータ&Android向けのパラメータ&ibi=iOS向けのパラメータ`の通りに非常に長いURLになります。
