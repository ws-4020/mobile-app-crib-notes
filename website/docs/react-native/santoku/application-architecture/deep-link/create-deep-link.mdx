---
title: ディープリンクの作成
---

ディープリンクURLの作成には[React Native Firebase Dynamic LinksのAPI](https://rnfirebase.io/dynamic-links/usage#create-a-link)を利用し、共有しやすいように[短いDynamic Link](https://firebase.google.com/docs/dynamic-links/android/create)を使用します。

### ディープリンクURLの構成

ディープリンクURLのドメインには、アプリケーションに関連付けたドメインを使用する必要があります。
またコンテンツリンク（`link`）はURL形式で、以下のように構成を決めて妥当なURLかどうかを検証できるようにします。

本アプリケーションではFireabseの`page.link`ドメインを利用するためディープリンクURLの構成は以下のとおりです。

|項目|APIのパラメータ|値|備考|
|:-------|:-|:--|:---|
|ディープリンクURLのプレフィックス|domainUriPrefix|アプリケーションに関連付けたドメイン|`https://santokuapp.page.link/`|
|コンテンツリンク|link|screenで画面、keyとvalueでパラメータを指定|`https://jp.fintan.mobile.santokuapp/`|
|iOSのフォールバックURL|ios.fallback|iOSアプリをインストールするときのURL|App StoreやTestFlight、DeployGate|
|バンドルID|ios.bundleId|アプリケーションのbundleId|`jp.fintan.mobile.SantokuApp`|
|androidのフォールバックURL|android.fallback|AndroidアプリをインストールするときのURL|Google Play、DeployGate|
|パッケージ名|android.package_name|Androidのパッケージ名|`jp.fintan.mobile.santokuapp`|
