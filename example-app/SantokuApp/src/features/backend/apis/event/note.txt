

backendCustominstance
引数でheadersを指定することができる
このときgetDefaultAxiosConfigに指定したheadersが消える

自動生成された段階で引数headersが指定されているAPIもある
  POST, DELETE   {'Content-Type': 'application/json'} 
  PUT   {'Content-Type': 'multipart/form-data'}

→　同じフィールドはオブジェクトの中身まで見てマージするようにしたい

obj1 = { headers:{'a':'a', 'b':'b'} }
+
obj1 = { headers:{'a':'a', 'c':'c'} }
↓
obj3 = { headers:{'a':'a', 'b':'b', 'c':'c'} }

===
obj3 = Object.assign({}, obj1, obj2 ) // 第一引数が変数の場合、それを書き換えてしまうため空のオブジェクトを指定している
